# 独立设计师智能助手 MVP需求说明

## 核心目标
构建一个基于n8n自动化和RAG知识检索的智能设计工作台，帮助独立设计师从灵感整理到生产落地的全流程效率提升，减少返工和决策成本。

---

## 具体场景

### 场景1：设计草图智能分析
```
1. 用户上传设计草图或输入设计描述（风格、廓形、面料需求）
2. n8n触发自动化工作流，调用RAG知识库检索相似案例
3. AI分析设计的工艺可行性和潜在风险点
4. 返回风险评估（高/中/低）和可行性分数（0-100）
5. 显示面料推荐清单、工艺建议和成本预估
6. 提供详细的设计优化报告和供应商推荐
```

### 场景2：专业知识问答
```
1. 用户在聊天框输入设计相关问题（工艺、面料、趋势等）
2. 系统基于RAG知识库检索相关专业文档内容
3. Claude AI结合检索结果生成专业解答
4. 返回格式化的知识解答和实用建议
5. 支持连续对话和上下文理解
```

### 场景3：灵感收集和整理
```
1. 用户输入关键词或选择预设风格类别（复古、运动、极简等）
2. n8n自动化工作流从WGSN、Instagram、小红书等平台收集相关素材
3. AI按颜色、材质、廓形等维度智能分类整理
4. 生成主题明确的灵感拼贴板和色彩方案
5. 显示趋势分析报告和市场预测数据
6. 支持素材收藏和个人灵感库管理
```

### 场景4：供应链智能对接
```
1. 用户输入面料需求（材质、颜色、克重、预算等）
2. n8n触发供应商数据库查询和价格对比工作流
3. 系统基于历史数据评估供应商可靠性和质量等级
4. 返回匹配的供应商清单和综合评分（0-100）
5. 显示价格对比、交期信息和质量评级
6. 提供一键联系和订单跟踪功能
```

---

### 前端交互
- 左栏：设计分析和灵感整理功能
- 右栏：知识问答功能
- 响应式设计，支持移动端
- 简洁现代的工作台界面风格
